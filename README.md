This repository contains the **SemaDoc Data Catalog** — a self-contained, static documentation viewer for Power BI Semantic Models generated by [SemaDoc](https://semadoc.io).

It ships two pages:

| Page | Path | Description |
|------|------|-------------|
| **SemaDoc Data Catalog** | `/en/docs-example` | Searchable catalog of all documented Semantic Models with sidebar, workspace filter, and full-text search |
| **SemaDoc Standalone Viewer** | `/en/viewer` | Client-side viewer — upload your Markdown files and render them directly in the browser, no server required |

## ✨ Features

- Full-text search across all models, tables, measures and documentation sections
- Tabbed model detail view (Model Overview, Tables & Columns, Measures, Import Logic, Calculation Items, UDFs, Security)
- Mermaid diagram rendering, DAX/Power Query syntax highlighting (via CDN)
- Dark mode, Zoom, Print, HTML export
- Static HTML only — no backend, no database, no runtime dependencies

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm

### Install & Run

```bash
npm install
npm run dev       # Dev server at http://localhost:4321
npm run build     # Build static site → dist/
npm run preview   # Preview built site locally
```

### Adding a Semantic Model

1. Create a subfolder under `src/data/markdown/` using the model's display name (e.g. `My Sales Model`)
2. Add the standard Markdown files generated by SemaDoc:
   - `model_documentation.md`
   - `table_documentation.md`
   - `measure_documentation.md`
   - `import_logic_documentation.md`
   - `calculation_item_documentation.md`
   - `udf_documentation.md`
   - `security_documentation.md`
3. Run `npm run build` — the model appears automatically in the catalog

## 🏗️ Tech Stack

- **Framework:** [Astro](https://astro.build/) (static site generator)
- **Styling:** Tailwind CSS 3.x
- **Language:** TypeScript
- **Output:** Static HTML (`dist/`) — host anywhere (webspace, S3, Azure Static Web Apps, …)
